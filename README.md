# Amber Wang Portfolio

Apple-inspired Streamlit portfolio website for internship applications.

## Positioning

This portfolio presents Amber Wang as an accounting student with strengths across:

- Audit and accounting workflow
- Business research and consulting-style analysis
- AI x finance and AI-assisted workflow building
- Product and vibe coding projects

## Features

- Recruiter-friendly landing page
- Modular About, Projects, AI Lab, Resume and Contact sections
- Filterable project gallery
- Four-step logic map for each project: Problem -> Method -> Output -> Value
- Resume and project PDF placeholders
- External GitHub and demo links

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Asset Paths

Put the latest resume here:

```text
assets/resume/resume_latest.pdf
```

Put project PDFs here:

```text
assets/projects/
```

The app will show placeholders until those files are added.
